Lennard-Jones source code programme description

This programme simulates the motion of a system of point particles which interact via the pairwise Lennard-Jones Potential:
U(ri,rj)=4*{1/r**12-1/r**6}, where r=|ri-rj|

------------------------------------------------------------------------------------------------------------------------------------------

IMPORTANT: the input and output files of the programme are written in the command line in the following order: 


Ar(g)_input.txt Ar(g)_XYZ.xyz Ar(g)_energies.dat Ar(g)_rdf.dat Ar(g)_MSD.dat

Ar(s)_input.txt Ar(s)_XYZ.xyz Ar(s)_energies.dat Ar(s)_rdf.dat Ar(s)_MSD.dat

------------------------------------------------------------------------------------------------------------------------------------------

The following system of reduced units is adopted for the simulation input and output:

Units of distance = σ = 0.3345 nm for Argon (1)
Units of energy = ε = 125.7 k_B for Argon (1)
Units of mass are such that the mass of the atom being used is 1
Units of time = σ * 1/ε^0.5
Unit of density = σ^-3
Units of temperature = ε/k_B

------------------------------------------------------------------------------------------------------------------------------------------

The initial conditions of the simulation are held in an input file called Ar(s)_input.txt/Ar(g)_input.txt which is of the following format:

# dt    Numstep
AA    BB
# N     T    rho
CC DD EE

For the Argon gas (Ar(g)_input.txt): N=30, rho=0.05, T=1.0, dt=0.005, numstep=2000
For the Argon solid (Ar(s)_input.txt): N=32, rho=1.0, T=0.1, dt=0.01, numstep=1000

The programme reads in the values of dt, Numstep, N, T and rho and passes them as arguments to the appropriate functions. The 
initial velocities and positions of particles are generated by the provided md_utils programme. 

------------------------------------------------------------------------------------------------------------------------------------------

Three output files are produced in the programme:

-> Particle trajectory XYZ file: Ar(g/s)_XYZ.xyz

N

N

... ...

MSD(sigma**2) time(sigma*1/epsilon**0.5)
0.0 0.0
... ...
